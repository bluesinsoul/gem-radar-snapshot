# Gem Radar Feed

Machine-written daily snapshot of **fundamentals-first low-cap-gem** signals ($5M–$250M),
distilled from the local `CoinResearch/GemRadar` pipeline (gems.db: CoinGecko developer/sector +
DeFiLlama protocol fees/TVL/revenue, over the `longshot_screener` price/supply/coil backbone).
Consumed by the **Gem Radar** cloud routine, which can't reach those APIs directly.

Raw signals only — **no composite score**. Gates qualify, Divergences are early-tells, Traps flag
distribution; the longshot coil score is the timing overlay. `null` = NO DATA (not zero). Overwritten daily.
